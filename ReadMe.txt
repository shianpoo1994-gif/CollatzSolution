Collatz – Testaufgabe



Benutzung
- Eine natürliche Zahl n (dezimal) eingeben
- Berechnung starten
- Ausgabe:
  * Anzahl der 3k+1-Operationen (ungerade Werte, ohne 1)
  * Anzahl der /2-Operationen (gerade Werte)
  * Maximales Element der Folge

Beispiel
Eingabe: 7
Anzahl der 3k+1-Operationen: 5
Anzahl der /2-Operationen: 11
Maximales Element: 52

Build (src)
1. "CollatzSolution.sln" in Visual Studio öffnen
2. Projekt "Collatz.Api" als Startprojekt festlegen
3. Build / Run
